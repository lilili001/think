这是默认主题default下User控制器的index文件
user：<{$user}>
data_name:<{$data['user']}> data_email:<{$data.email}>
obj_name:<{$obj->name}> obj_name:<{$obj:name}>


前面输出变量，在后面定义
<{$user|mb_substr=0,3,'UTF-8'}>

这句话的作用是将$date时间戳转化为日期格式，传参 $date 和 "Y-m-d H:i:s"两个参数,###的作用是将$date作为第二个参数
<{$date|date="Y-m-d H:i:s",###}>

多个函数用"|"隔开即可
<{$user|mb_substr=0,3,'UTF-8'|md5}>

PS：如果你觉得以上写法需要在脑海里二次翻译，太过于麻烦，那么可以用以下的格式写法：{: 普通php写法 }
<{:md5(mb_substr($user,0,3,'UTF-8'))}>

当传递过来的变量如果没有值的时候，模版提供了默认值输出功能。
<{$nodate|default='没有数据！'}>

我们可以在模版中使用运算符，包括对“+” 、 “-” 、 “*” 、 “/” 、 “%” 、 “--”和“++” 的支持。
<{$num+10}>
<{$num+getNum()}> getNum函数在Weibo/Common/Common/function.php中

模版还支持三元运算符： <{$withdata ? '$withdata有值' : '$withdata无值'}>
